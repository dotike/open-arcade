### To install

* mkdir Git
* cd Git
* git clone <THIS REPO>

* cd ~/Git/arcade/lib/python
* pip3 install ./arclib
* pip3 install --upgrade ./arclib
* pip3 uninstall arclib

* cd ~/Git/arcade/docs
* pdoc --html arclib
* pdoc --html --force arclib


### To use

* from arclib import < sub-module > as < alias >
